Files Control Tool

* **Завантаження XML**
* **Конвертація**
* **Зберігання**
* **Управління**
* **Фільтрація**

Технологічний стек

* **Java 21+**
* **Spring Boot 3** 
* **H2 Database**
* **Thymeleaf** 

Як запустити проєкт

1. **Клонуйте репозиторій**
2. **Запустіть додаток**
3. **Відкрийте браузер та протестуйте на базовому порті 8080**

Примітки

* **Формат назви файлу**: Для коректного парсингу файл обов'язково має називатися за шаблоном: `client_invoice_2026-05-11.xml`.
* **Локальне сховище**: Всі завантажені файли фізично зберігаються в папці `upload` у корені проєкту.
